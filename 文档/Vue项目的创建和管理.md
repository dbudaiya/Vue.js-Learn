## 使用Vite创建项目

```
$ npm init vite-app <project-name>
$ cd <project-name>
$ npm install
$ npm run dev
```

创建reate项目

```
$ yarn create vite-app <project-name>
$ cd <project-name>
$ yarn
$ yarn dev
```

> try `npm init vite-app --template react` or `--template preact`.
